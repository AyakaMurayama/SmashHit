# SmashBall



`無限に跳ねるスポンジボールを上層階へ運んでいくゲーム`
ジャンル:ワンタップアクションゲーム
プラットフォーム:iOS アプリケーション
ターゲット層:手軽にゲームをしたい人
開発環境:Unity

無限に跳ねるスポンジボールを上層階へ運んで行くゲーム。
ボールがインクにぶつかるとスポンジがインクを吸って重くなってし
まい、跳ねる高さが低くなってしまう。
操作はタップでジャンプ、ジャイロ機能でボールの方向を決定させる。
